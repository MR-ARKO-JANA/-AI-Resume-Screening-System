# Contributing to AI Resume Screening System

Thank you for considering contributing to this project!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/-AI-Resume-Screening-System.git`
3. Install dependencies: `npm install`
4. Create a `.env` file based on `.env.example`
5. Start the server: `npm start`

## Development Guidelines

- Follow the existing code style (2-space indentation)
- Add JSDoc comments for new functions
- Test your changes locally before submitting a PR
- Keep commits atomic and well-described

## Project Structure

- `backend/` - Express.js server, routes, controllers, models
- `frontend/` - HTML, CSS, JavaScript client files
- `__tests__/` - Jest test files

## Reporting Issues

Please use GitHub Issues to report bugs or request features.
